### 1.9 운용 시퀀스

```
[Phase 1: 서칭]
SEARCH-1 (62°) → 타겟 탐색 (넓은 영역)
              → 비항성체 발견 (별 카탈로그 비교)
              → 위성 자세 조정 명령
                        ↓
[Phase 2: 획득]
SSA-CAM (3.6°) → 타겟 FOV 진입
              → 정밀 추적 시작
              → AI 거리 추정
                        ↓
[Phase 3: 접근]
SSA-CAM + 스테레오 → 거리 융합
                  → EKF 상태 추정
                  → 150m까지 접근
                        ↓
[Phase 4: 근접 촬영]
SSA-CAM → 150m에서 고해상도 촬영
       → 3U CubeSat 146 px 해상도
       → 타겟 식별 및 상태 분석
```

## Part 2: 추진 시스템

---

### 2.1 시스템 개요

| 항목 | 사양 |
|------|------|
| **명칭** | ECSP (Electrically Controllable Solid Propellant) |
| **형식** | 전기제어 고체 추진제 |
| **용도** | 근접 운용 (Proximity Operations), 자세 제어 |

---

### 2.2 전력 요구사항

| 항목 | 사양 |
|------|------|
| **피크 전력** | 400 W |
| **피크 전류** | 5 A |
| **구동 전압** | 80 V |
| **분해 시간** | 2-3분 |
| **운용 모드** | 간헐적 (가스 분해 → 탱크 축적 → 분사) |

---

### 2.3 전력 아키텍처

```
┌─────────────────────────────────────────────────────────────┐
│                    추진계 전력 체인                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   EPS (Battery Bus: 8-16V)                                  │
│       │                                                     │
│       │ 10~20W (제한된 충전 전류)                            │
│       ▼                                                     │
│   ┌───────────────────────────────────────┐                │
│   │         PPU (Propulsion Power Unit)    │                │
│   │                                        │                │
│   │  ┌──────────┐    ┌──────────────────┐ │                │
│   │  │ 충전기    │───▶│ 슈퍼캡/울트라캡   │ │                │
│   │  │ DC-DC    │    │ 뱅크             │ │                │
│   │  └──────────┘    └────────┬─────────┘ │                │
│   │                           │            │                │
│   │                  ┌────────▼─────────┐ │                │
│   │                  │ 부스트 컨버터     │ │                │
│   │                  │ 12V → 80V       │ │                │

│                               ▼                             │
│   ┌───────────────────────────────────────┐                │
│   │         가스발생기 (Decomposer)         │                │

│   └───────────────────────────────────────┘                │
│                               │                             │
│                               ▼                             │
│   ┌───────────────────────────────────────┐                │
│   │            가스 저장 탱크               │                │
│   └───────────────────────────────────────┘                │
│                               │                             │
│                               ▼                             │
│   ┌───────────────────────────────────────┐                │
│   │       솔레노이드 밸브 (8개)             │                │

│   └───────────────────────────────────────┘                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

---

### 2.4 운용 개념

| 단계 | 전력 | 시간 | 설명 |
|------|------|------|------|
| **충전** | 10-20W | 수십 분 | EPS → 캐패시터 뱅크 충전 |
| **분해** | 400W (피크) | 2-3분 | 캐패시터 → 가스발생기 가열 |
| **축적** | - | - | 생성 가스 → 탱크 저장 |
| **분사** | 저전력 | 펄스 | 솔레노이드 밸브 → RCS 노즐 |

**핵심: 탱크 한번 채우면 여러 번 분사 가능**

---

### 2.5 RCS 구성

| 항목 | 사양 |
|------|------|
| **노즐 수** | 8개 |
| **배치** | **Canted (경사 배치)** |
| **제어 자유도** | **6-DOF (3축 자세 + 3축 병진)** |
| **제어 방식** | 솔레노이드 밸브 |
| **용도** | 자세 제어 + 병진 기동 |

```
         ┌─────────────────────┐
         │      6U CubeSat      │
         │                      │
         │    ╲1          2╱    │   ← 상단 Canted
         │     ╲          ╱     │
         │      ╲        ╱      │
         │                      │
         │      ╱        ╲      │
         │     ╱          ╲     │
         │    ╱3          4╲    │   ← 하단 Canted
         │                      │
         └─────────────────────┘
         
         (전면/후면 각 4개씩, 총 8개)
         
         Canted 배치로 6-DOF 제어:
         - Roll, Pitch, Yaw (자세)
         - X, Y, Z (병진)
```

---

### 2.6 보드 구성

| 보드 | 역할 |
|------|------|
| **PPU** (Propulsion Power Unit) | 캐패시터 충전, 승압, 전력 공급 |
| **PCU** (Propulsion Control Unit) | 가스발생 제어, 밸브 제어, 센서 모니터링 |
